I'm open to recommendations on a clever mainlist name. I'm choosing Triple Threat: Intracommunal Men of Obscurity. Because more Austin Powers references.

To head off a few questions:

The idea is that the collapsible is meant to represent you heading out to Oak Grove and reading their item summary. The 'real' document is not actually in the main SCP database; I would have used the main article as just a splash page, but I figured it would be much simpler to just have it all on one page.

Alright, I have to ask

So how am I reading about it then?

The 'requires an in-person visit' doesn't hold up against the anomalous effect of preventing information regarding the skip in any format from leaving the town. Unless I personally went to the town to read this, I shouldn't be able to have this information, right?

I think you built too perfect of a prison here. I'm going to re-read to see if I missed something, but simply because it's (mostly) technically sound and well written I will be no voting for now.

Edit: Upon rereading I see that I forgot the note that some very general information is able to be communicated, but I think that explanation is frankly a little too convenient. The information in this article isn't general at all, there's a lot of very specific info (pretty much any info you'd need to write a skip) so that by the end it didn't even seem like there were any antimemetic properties whatsoever.

I think a LOT of this info would need to be removed for them to be convincingly anomalous, because right now I know everything I would need to know about these kids, as far as I can tell.
