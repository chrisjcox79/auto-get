The thing is, there's a guideline called the . 9 times out if 10, it's an accurate indicator for what your object class should be.

The Locked Box Test is more or less as follows:

If you lock it in a box and it stays there, it's Safe.
If you lock it in a box and need to check up on every so often it to make sure it's still there, it's Euclid.
If you lock it in a box, and the only thing keeping it in the box is twine and prayer, it's Keter.
If it is the box, it's Thaumiel.

Strictly speaking, the Foundation is required to actively maintain two nightclubs and make sure they're popular venues, or the SCP breaches containment and they gotta set up or buy out more nightclubs. They can't forget about the SCP and assume it'll sit there indefinitely, they need to be actively working on its containment to make sure it stays put.

This is a solid Euclid, no two ways about it.

I want to tell an everyday story in the SCP universe, so that means I want 3119 to be Safe.

Promising start for a new writer. I'm afraid I downvoted based mainly on the lapses in clinical tone. Use of , these represent too large a break from the existing unwritten style guide. I tend to be quite conservative with this sort of thing, I'm afraid.

Speaking to the narrative: the entities are meant to be broadly sympathetic, and that's conveyed effectively, but I couldn't truly feel them. They're incorporeal emotionally as well as physically, somehow. The passage at the end with the  a little more description of their reactions like that would have helped bring them to life. I can't help but wonder if they'd have felt more real if they had some kind of actual projected avatar – if the person whose mind was being occupied experienced some distinct image or persona that they could recount to us in the document.

But I'm looking forward to seeing you write more here. You've got a distinctive approach to scip-writing that I want to see more of.
