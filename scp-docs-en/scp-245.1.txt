Hello. This is an SCP that is only describable using interactive media. I utilized RPG Maker to create this, and there are 4 endings.

Edit: SPOILER WARNING: This is a video of me playing through all 4 of the game's endings: https://www.youtube.com/watch?v=zlEV1nhwVWA

Spoilers below:

+block

–block

The bad ending(not doing all the quests properly) gives you the password to the file on the wikipage, but you can get it in the good ending (completing all  of the quests properly) if you wait around in the  long enough.

After you've done that, restart the game and beat the green cloaked guy to the first control room. That's the main secret ending, and gives you a full description at the end of it.

If you're really adventurous, there's one more secret ending, but since it's not necessary for the story, I won't spoil it for you.

Currently there are a number of names you can input at the beginning of the game to get easter eggs as well. And a few messages spread throughout the game itself. Have fun!

I got lots of help and testing on this. Pretty sure I dinged all the testers in the credits, but special thanks to Decibelles, TwistedGears, and Waterfire. Additionally thank you to SunnyClockwork for the beautiful artwork I used in the game.

Note: My original SCP-245 was moved to the SCP-2623 slot with specific explicit permission from the administration given the nature of the project I put into the SCP-245 slot. It should not be taken as a precedent of any kind as to permission to do any such action in the future.

Also! If you play this and spot a bug, mention it here. I'll fix it as soon as possible. I've run through this myself a hundred times or more, but I'm sure I or my testers missed something.

And I've requested that the page be locked so no one can edit it and mess with the links. If you're wondering why it's locked.

Edit: Until I was halfway through making it, I was unaware that this wouldn't run on Macs (or even that Macs can't run EXEs). Despite that, I have been told by other users that a windows emulator can run this program no problem. Suggested emulators such as  have been mentioned.
