25 Jan 2014 04:07

Eh, there's a number of writing problems with this. First off, this is number soup (soup would be interesting to try with this, actually). This line:

This also applies to the ingestion of SCP-1842-3.
附加记录：    

Seems to imply the resulting foods are actively toxic, rather than just being nutrition-free. Which strikes me as implausibly nonfunctional. Wondertainment doesn't release stuff that would harm children if used as directed. I can believe the -2 as being harmful if swallowed, though not anomalously so, but it does not match the universe to have -3 be so. It's unclear how exactly the last test worked, if it can only produce food items.

Tests have indicated that the output of SCP-1842-2
附加记录：    

This line reads as off. -2 isn't something that has output.

The Foundation Toxicology lab has confirmed
附加记录：    

This is phrased poorly. Why not ?

This is a solid premise, but the execution is flawed, and it never goes anywhere. Downvoting for now. If you can fix these issues, I'd neutral vote, and if you can have the article go a more sensible and interesting direction than the gun test, I'd probably upvote.

Eh, sorry, this still doesn't work for me. You've fixed some things, but there's still awkward phrasing. Describing the non-toxicity of -3:

However, instances of SCP-1842-3 possesses no nutritional value, and should not be consumed.
附加记录：    

However, once SCP-1842-2 is converted into SCP-1842-3, it is no longer toxic. However, it still retains no nutritional value.
附加记录：    

These shouldn't both be there. I'd rearrange things to have the paragraph describing -2's toxic effects before you get into the main anomaly. It is burying the lede, but it prevents you from bouncing back and forth so much between topics. You also way, way overuse the word .

The changes to the test log are okay, but there's a lot of logical issues with the trebuchet. You establish with the arm that it only produces stuff that the user perceives to be food, and though cannibal cults may exist, I don't think trebuchetvorous cults do. Plus, considering the physics of a trebuchet, a functional one made out of foodstuffs as described would also be impossible. For that matter, making one out of play-dough that doesn't fall apart would be quite a feat.
