▷Info

This is inspired by Rimple.
The researcher notes in Addendum 3 are not the end of the article; keep scrolling down to find the rest.

NOTE:  The  section is functional and allows you to type into its text box. This does not work on most mobile browsers and select desktop browsers. If it fails to work, try pressing space first.
The HTML code for Addendum 3's  is adapted from Westrin gave some help modifying it for use in this article.

The link text for each collapsible references specific World of Warcraft quotes (three in total) from the character Il'gynoth (Source ).
We received permission from both Zyn to use their likenesses in this article. References to the SCP Wiki Forums and Senior Staff procedures are not there to mock or make fun. They exist to take this meta-story to its logical conclusion.
