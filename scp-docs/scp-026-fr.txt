                        

项目编号：  SCP-026-FR

威胁等级：  黄 ●

项目等级：  Safe

特殊收容措施：  SCP-026-FR的副本必须存放在为此目的提供的CD套中。由于光盘数量众多，口袋必须存放在2x2x2米的安全区域内。由于SCP-026-FR仅在人类精神上发挥作用，只需要研究人员了解其效果就足以令其他人员不要试图复制。将为监视分配摄像机和警卫，以识别和阻止任何违法者。

描述：  SCP-026-FR是一组异常物体，在通常情况下包含一个名为“nu-metal”乐队的最新专辑的记录，名为█████████ ████，由十二个单曲组成，总共39分钟46秒。在基金会调查之后，这些记录将成为专辑的所谓“先行体验”版本，并已商业化名义分发了666份。迄今为止，所有项目都是从其所有者那里寻获并没收的，之后，这些所有者均已被给予B级记忆清除。

只有在可播放音乐的媒体上播放时，才会显示对象的实体影像。那些能够听到音轨的人将会毫无区别地受到影响（详见附录-026-01-FR）。从播放第一首曲目的那一刻开始，就不再可能对听众进劝离，此时听众被称为SCP-026-FR-1。如果试图以任何方式阻止广播，在场的SCP-026-FR-1，将对进犯者作出暴力反应。如果不在场，则发生骚乱，直至完成“演唱会”为止。因此强烈建议等到最后一首曲目停止以回收光盘。据观察，不同音乐不同列表可能播放出不同效果。

在播放第一章节时，SCP-026-FR-1的实体将会聚集在SCP-026-FR周围。原先不喜爱此类风格音乐者，会逐渐表现出对SCP-026-FR的行为的兴趣。名为“领导者”的实体将出现在SCP-026-FR-1群体中。 他们将比其他人更有活力，并将在人群中传播。 已经确定只有那些已经参加过“金属”或“朋克”式音乐会的人才会成为领导者。

从第二章节开始，项目将开始活动，直到第十二首曲目结束。以下以非详尽的方式列出了数个事实：

随后各“领导者”将开始尖叫 ，随后人群中将会从中分开，留出一个空位，领导者将站在其中，开始作淫秽的手势和令人不适的舞蹈。在10秒之后，SCP-026-FR-1的两个部分将在这个区域面对，相互冲撞，出现不明原因且难以置信的暴力和冲突。数分钟后，SCP-026-FR-1将停止冲撞，并且它们中的每一个都将返回原位，在它们之间留下更多空间。

领导者会突然对其中一人实施推搡。由于不明关系，根据组成SCP-026-FR-1的受试者总数而变化，所有人将开始不遗余力地互相推搡，并形成一个圆圈，划定为“擂台”。每当参与该事件的一个实体从该区域弹出并与圆的轮廓接触时，都将被拉回该圆内。这一事件最初被早期观察的研究员描述为“残忍和虐待狂才会做的事情”，而却被习惯于该音乐风格的工作人员解释为“这只是为了娱乐和安全”。其中两个实体在任何情况下都可避免任何致命的践踏。这种现象似乎是定时发生，每首歌曲重复至少一次，间隔时间保持不变，持续时间不少于4分钟。

一个领导人将站位于一点上，他们将通过摇动他们的手臂并疯狂地挥舞他们的脚来开始表演所谓的“舞蹈”。其他人将逐渐加入他们。与之前的活动不同，他们不会特别渴望与其他人接触，但此类舞蹈需要很大的空间。

笔记：  - Dr. ████████ (数日后该博士因受袭击住院)。

两名领导者突然开始自言自语并大喊大叫 。很快，一个圆形人群阵就会形成，但不仅仅是互相推搡或者跳舞，圆圈中的实例将在转弯时同时做出更多类似动作。

当上述事件都没有发生时，一些“领导者”会站在人群中并被它带走，同时开始尖叫，用食指和小指伸向天空其他手指折叠在手掌上，直至到达SCP-026-FR面前。然后它们将被SCP-026-FR-1击倒并推回原位。

在演唱会期间，SCP-026-FR-1的每个实体都会被戏弄与表现得精力充沛。通常会向各个方向猛烈摇头，并且通常显示被认为是他们的独特标志，即食指和小指向天空，通常被本部研究人员称为“魔鬼之角”。

一旦CD停止播放，SCP-026-FR-1组成部分将逐渐恢复原本意识，但不会记住之前任何事情。如果有人受伤，其则会认为自己的伤势来自日常事故。人群将分散并停止一切活动。只有名为“领导者”的实体才可记住演唱会，并希望再次触碰SCP-026-FR。机动特遣队必须迅速采取措施，并对所有目击者采取B级记忆删除措施。

附录-026-01-FR :  SCP-026-FR的无意听众通过无线电或电视听取音乐后将采用相同的行为，但不会试图复制这类音乐。

采访记录-026-01-FR :  在SCP-026的一个非收容行动结束时，现在的特遣队得以采访其中一个“领导者”，以便更容易探明他们出现的可能性。（该实体将被命名为“SCP-026-0”）

特工 ███████ :  天气真好，请说说你的身份是啥吧

SCP-026-0 :  我的名字是[数据删除]。该死的这张专辑真是个好东西！如果我现在把它撤回去，这会让我感到非常困扰。

特工 ███████ :  你能否至少等到这次采访结束？

SCP-026-0 :  噢，嘿，对不起。就是这样这太疯狂了，我从来没有过如此多姿多彩的生活！

特工 ███████ :  *咳嗽*让我们重新开始，你能告诉我你是如何成功吊起所有这些人的胃口的吗？

SCP-026-0 :  好吧，我已经做了很多演出，然后这个乐队简直棒极了，堪称史诗！显然现在很多人都不习惯，好吧，但是我们向他们展示了如何做到这一点。哈哈哈！每个人都紧随其后，紧随潮流！

特工 ███████ :  “我们”？你还知道有其他领导者吗？

SCP-026-0 :  不知道，但肯定有一个真正的金属头头，不然我们也没法把乐队做得这么大。

特工 ███████ :  “金属”？

SCP-026-0 :  好耶！人人都对我们抱有仰慕之情。你听说过金属吗？我更有理由让你听的专辑，你会发现它不是我们收音机里流行的“流行音乐”！

特工 ███████ :  采访结束后，如果你愿意，我还有两三个问题要问你而且，也许你得意识到，在你的破音乐会上，已经有不少于四人受伤？

SCP-026-0 :  有人受伤？好吧，即使我们盯着也总是可能发生。但事实并非如此，他们并没有因此而说什么。你看，他们会因为受伤而责备我们吗？不，我不认为这只是一种情绪麻痹而已。必须停止这场闹剧，我们又不是神经病。金属，是一种心态，是我们分享和生活的东西！不是大规模谋杀！好吧结束了，我现在可以发行这些专辑了吗？
