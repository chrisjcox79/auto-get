                        

项目编号:  SCP-2029-JP

项目等级:  Safe

特殊收容措施:  各SCP-2029-JP个体样本需分别收容于使用山添式灵体隔断壁的收容室。因使用SCP-2029-JP的实验现在处于完全冻结状态，故在高级研究员向O5会议提交申请文件后，需获得过半数成员的许可方能进行实验。神津博士所负责的后续收容工作将继续进行。

描述:  SCP-2029-JP本体为由神津博士负责研发制作的哈特曼灵体摄影机的初期机型。目前已被确认收容的机器样本有5台。自19 ██/██/██之后，该机型除实验目的之外均无任何使用记录。

SCP-2029-JP的原型机受第七次超自然世界大战之后基金会内部对灵体研究的发展影响，于19年在神津博士主导的特殊部队中以获得灵体实体稳定图像记录为目制作而成。SCP-2029-JP内部内置灵体Exporasm溶液作为灵体样本。利用此样本与作为对象的灵体附着位点的差值进行投影，从而完成灵体成像技术。详细内容请参照相应论文。

SCP-2029-JP因未知原因被赋予了上述论文中未记载的抗破坏性质。因此无法对其内部结构进行解析，作为Exporasm溶液被使用的灵体样品的提供源也同样不明。现在基金会内部使用的哈特曼灵体摄影机是由伊格纳兹·哈特曼客座教授参考SCP-2029-JP的异常性质以及上述论文后制作而成的。

使用SCP-2029-JP进行对精神实体的摄影时，摄影者会感知到类似男婴啼哭的杂音，或类似不明生物吞噬杂物的异常声响。且摄影者会在在无知觉情况下受到多次擦伤。这种现象不管作为拍摄对象的实体性质如何都会发生。另外，作为该项目拍摄对象的灵体在多数感知到SCP-2029-JP的情况下，均会表现出类似困惑/疑惑的情绪。除以上几点异常现象外，SCP-2029-JP的摄影精度与现在使用的哈特曼灵体摄影机没有明显的差别。

制作SCP-2029-JP的神津博士对上述异常性质没有明确正面的回答，根据相关报告显示，神津博士在执笔相关论文后，其本人与其家庭成员全部失踪。

附录1 - 19██/██/██:  根据后续调查显示，神津博士的长子浩太郎（当时9岁）可能早在神津氏一家失踪之前便下落不明。

附录2 - 19██/██/██:  对SCP-2029-JP，使用现役哈特曼灵体摄影机进行了交互实验。结果表明，SCP-2029-JP以叠加态形式附着有灵魂实体（以下称SCP-2029-JP-1）。经鉴定，SCP-2029-JP-1的灵体呈像与神津博士之子浩太郎的外貌相似度超过90%。因此推断SCP-2029-JP内部使用的Exprise溶液的来源为浩太郎。收到上述发现后，实验组尝试与SCP-2029-JP-1进行进一步交互。但其对所有语言及可通信设备均未做出反应。

以下是作为附属实验进行的对精神实体采访。采访者为对SCP基金会无明显敌对意识的灵魂实体SCP-2996-JP-EX-1。

音频记录2029-JP-01 - 日期 19██/██/██ 

采访者 : SCP-2996-JP-EX-1

对象 : SCP-2029-JP-1

«记录开始» 

(前半部分省略）

指挥部 : 现在开始尝试进行接触，SCP-2996-JP-EX-1，准备好了吗。

SCP-2996-JP-EX-1 : 没问题。

(投放SCP-2029-JP)

指挥部 : 那接下来，尝试和它对话吧。

SCP-2996-JP-EX-1 : 呃，初次见面.?

[SCP-2996-JP-EX-1表现出困惑的情绪，沉默。记录到SCP-2029-JP附近发出来源不明的水声]

指挥部 : SCP-2996-JP-EX-1，发生了什么？

SCP-2996-JP-EX-1 : 那个，博士们不是说对方是幽灵吗？

指挥部 : 是，我告诉过你对方是灵魂实体。

SCP-2996-JP-EX-1 : 这么说来，大概是对它有什么误会吧。

[SCP-2029-JP周围产生实体碰撞声。之后的调查显示，实验场地的四壁均出现细小裂痕］

指挥部 : 等等，你的意思是，它不是灵体吗？

不明实体 : 出去。

SCP-2996-JP-EX-1 : 对不起我必须走了。

«记录中断» 
附加记录：    

不明实体发声后，SCP-2996-JP-EX-1通过采访室向外部逃走。同时，参与监视的研究员产生分布性休克现象，受影响人员的大脑皮层受到损伤。其大脑部分的识别机能产生障碍。根据此实验，推测SCP-2029-JP附着灵的实体有被观测到的可能。

根据后续工作确认，出逃的SCP-2996-JP-EX-1产生一定混乱情绪，但没有较大损伤。以下是对SCP-2996-JP-EX-1的采访记录。

音频记录 2029-JP-02 - 日付 19██/██/██ 

采访者 : 井上博士

采访对象 : SCP-2996-JP-EX-1

«记录开始» 

井上博士 : 精神状态还行吗？如果不舒服的话随时都可以停止记录。

SCP-2996-JP-EX-1 : 没问题。

井上博士 : 好，那开始确认信息，首先，你在SCP-2996-JP里看到了什么？

SCP-2996-JP-EX-1 : 你们都说那个照相机里有幽灵附体，有确实算是有吧，之前博士跟我说对方是个男孩，但是它不是。呃，不对，我不知道它是不是。

井上博士 :有什么奇怪的地方吗？

SCP-2996-JP-EX-1 : 是啊，怎么说才好呢，虽然它说话方式不太好辨认，像是人的残骸。大概可以看出形状，但是到处都有削切的痕迹，还能看到露出的肌肉和脂肪，差不多就这样。

井上博士 :原来如此，对方的灵体受到了严重损伤难以辨认，这就是你困惑的理由吗？

SCP-2996-JP-EX-1 : 不，我不太清楚那个残骸背后的存在，我不知道那该说什么。它给人的感觉非常取走了。

井上博士 : 我不太能理解你的意思，你是说机械里装着的那个东西是只有灵的实体才能够识别到的，而且具有攻击能力的实体吗？

SCP-2996-JP-EX-1 : 我觉得它和普通的灵稍微有点有点不一样。与其说是被封入的，倒不如说它和这个机器是一样的东西。有点难解释吧，可是我总觉得它对我们这边没有什么兴趣。如果不是那样的话，早就不，应该不是。可能只是没有注意到而已。如果博士也能看到就明白了，不就是那样的东西吗。怎么说呢，好像在和我们不一样的地方，却在同一个地方。

不明实体 : 眼睛，堵住

SCP-2996-JP-EX-1 : 博士！快闭眼！

«记录中断» 
附加记录：    

根据后续调查显示，在记录中断后的34秒内，井上博士及SCP-2996-JP-EX-1被不明原因造成分布性休克导致晕厥及暂时性视神经失调。意识恢复后，虽然没有检测到明显的大脑功能的损伤。但事后二人均声称未曾有过与SCP-2996-JP有关的记忆。

为避免项目进一步对人员安全造成进一步影响，SCP-2029-JP和SCP-2029-JP-1的交互实验被无限期冻结。上述事件发生后，进一步观测报告显示处于SCP-2029-JP附近的人员会听到类似人类打喷嚏的杂音。

补充材料（来自此文档第██任编辑者：神津博士）  20██/██/██，我把新的SCP-2029-JP送去了81XW网站。经过细致的研究刨析后，可以断定SCP-2029-JP没有什么所谓的抗破坏性，所以出于调查需要对他进行了拆解。拆解结果显示他的内部除了人类眼球之外什么都没有。上述论文中记载的异常结构简直扯淡。但即使构造如此简单，他的性能也与其他SCP-2029-JP相同。

注：眼球的持有者不明。

针对此次补充材料编辑所进行的任何信息追踪工作均无有效进展，需注意的一点是，若神津博士尚未死亡，其年龄应为152岁。
